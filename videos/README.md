# Videos Folder

This folder contains video files for your portfolio projects.

## Supported Video Formats
- MP4 (recommended)
- WebM
- OGV

## Video Guidelines
- **File Size:** Keep videos under 50MB for web performance
- **Resolution:** 720p (1280x720) or 1080p (1920x1080)
- **Duration:** 1-5 minutes for best engagement
- **Compression:** Use H.264 codec for maximum compatibility

## Portfolio Videos
- `metamaterials-transformation.mp4` - Metamaterials transformation process
- `metamaterials-testing.mp4` - Material testing demonstration
- `thesis-defense.mp4` - Thesis defense presentation
- `thesis-demo.mp4` - System demonstration

## How to Add Videos
1. Place your video files in this folder
2. Update the portfolio markdown files with correct video filenames
3. Ensure video files are optimized for web viewing

## Video Hosting Alternatives
If your videos are too large, consider:
- YouTube (embed with iframe)
- Vimeo (embed with iframe)
- Google Drive (shared links)

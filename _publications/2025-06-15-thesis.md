## Overview
Photovoltaic (PV) output is highly volatile at minute-to-second scales because cloud evolution can rapidly reshape irradiance. This thesis addresses a key limitation of many sky-image-based forecasting approaches—**spatial ambiguity**—by building a **multi-stage, physics-aware pipeline** that turns ground-based sky imagery into PV-relevant, spatially interpretable inputs. The core idea is to go beyond “2D cloud prediction → PV output” and instead connect vision to physical projection: **geolabeled vision features → long-horizon cloud nowcasting → 3D cloud voxelization → site-level shadow/irradiance mapping → PV forecasting-ready signals**. This framing enables forecasting models to leverage not only temporal patterns but also physically grounded spatial cues (cloud position, sun geometry, and shadow placement).

## My Contributions
- Built a **geolabeled vision preprocessing** workflow (pixel-to-WGS84 mapping, sun-position estimation + solar masking, cloud contour extraction) and validated coordinate-level consistency.
- Designed and executed an outdoor experiment to construct a paired **shadow texture ↔ irradiance** dataset (shadow/no-shadow samples aligned with irradiance readings) to bridge vision outputs with PV-relevant supervision.
- Implemented and benchmarked multiple **cloud nowcasting model families** under a unified protocol, selecting architectures with stronger long-sequence generation capacity for downstream spatial modeling.
- Proposed and analyzed a **2D→3D→projection** extension (Pix2Vox-style voxel reconstruction + plant-region projection) as a path toward spatially explicit PV nowcasting.

## Methodology Pipeline
**Problem framing → Data engineering (vision + geolocation + sun geometry) → Dataset construction (shadow–irradiance supervision) → Semantic modeling → Cloud nowcasting benchmarking → 3D reconstruction concept → Projection to shadow/irradiance → PV forecasting-ready outputs**
